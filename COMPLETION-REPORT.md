### 3️⃣ سیستەمی بەرهەمی No-Code ✅

**فایلەکان:**
- `db.js` - کۆناڵی SQLite
- `schema.sql` - نیوێتی تێکڕاوی داتابەیس
- `api.js` - هەموو APIی پلاتفۆرم

**APIی دروستکراوە:**
- ✅ `POST /api/products/generate` - دروستکردنی بەرهەم
- ✅ `GET /api/products/store` - کۆگای بەرهەمەکان
- ✅ `POST /api/payment/webhook` - پڕۆسیسی پارەدان
- ✅ `GET /api/products/:id/activation-key` - وەرگرتنی کلیل
- ✅ `POST /api/ads/generate` - دروستکردنی آگهی خۆکاری
- ✅ `POST /api/products/external` - زیادکردنی بەرهەمی دەرەکی

## 🎯 ئەوە کاتێ دەتوانت تێسک بکرێت

### ١. سیستەمی زمان:
```
1. بکەوە ماڵپەڕە
2. زمانی سێیان هیل بکە (کوردی/عەرەبی/ئینگلیزی)
3. بڕوا گۆڕینی زمان کردە
4. پێنجەرە پیشتەوە بگؤڕە
5. لە localStorage بگێڕە (F12 → Application → LocalStorage)
```

### ٢. سیستەمی ئەجێنت:
```
1. 🤖 پوتەنی ئەجێنت خاپی بکە
2. نیشێکی هیلبژێرە (📚 پلانەکان، 🎨 دروستکردن، ئەتر)
3. ئەجێنت وتاری فێرکردنەوە بدات
4. پرسیار لە ئەجێنت بکە
```

### ٣. سیستەمی بەرهەم:
```
1. پڕۆمپت بنووسە
2. ناردن خاپی بکە
3. بەرهەم دروست دەبێت
4. کۆگادا دەردەکەوێت
```
