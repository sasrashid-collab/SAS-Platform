# 🚀 SAS-tech Platform - سیستەمی سێ زمان و ئەجێنتی ڕێنمایی

## 📖 خصوصیات سەرکی

### ✨ سیستەمی سێ زمان (3-Language System)

پلاتفۆرمەکە بەپێی کردار پیشتگیری لە **٣ زمان** دەکات:

| زمان | کۆد | چاپمان |
|------|------|--------|
| **کوردی** | `ku` | RTL (ڕاست بۆ چپ) |
| **عەرەبی** | `ar` | RTL (ڕاست بۆ چپ) |
| **ئینگلیزی** | `en` | LTR (چپ بۆ ڕاست) |

#### 🔄 بگۆڕنی زمان:
1. **سەرەتا** - کاتی دووچاویکاری، خوێندەر لە سێ زمان هیلدەبژێرێت
2. **ڕۆکاری سەرەکی** - پلیت بۆ گۆڕنی زمان لە سەرپەڕدا هەیە

#### 💾 ساتێندی:
- زمانی هیلکراو لە localStorage تۆمار دەبێت
- دوپات دێوچاویکاریدا، زمانی پێشوو بارکراو دەبێت

### 🤖 سیستەمی ئەجێنتی ڕێنمایی (Help Agent System)

ئەجێنتی زیرەک چالاک کە بتوانێت گێڕان و رێنمایی بدات لە هەموو ڕووی پلاتفۆرمدا:

#### 📚 بابەتەکانی ئەجێنت:

**١. پلانەکان (Plans Explanation)**
- وێنا کردنی سێ پلانی مختلیف
- ئامێزگاری ڕێ چاویی هەر پلانێک
- سنووری توانایی هەر پلانێک

**٢. دروستکردنی بەرهەم (Product Creation)**
- ڕۆژنامهی قۆل بە قۆل
- فێری بابەتی پڕۆمپت نیویسین
- وتاری ئۆتۆماتیکی کۆدی دروستکردن

**٣. فرۆشتن و دابەشکردنی پشک (Sales & Commission)**
- ڕێنمایی سیستەمی پەرەپێدان
- فێری دابەشکردنی ٢٥% - ٧٥%
- فێری سیستەمی کلیلی ئەکتیڤکردن

**٤. سیستەمی بەدیهی (Default Responses)**
- وتاری دۆستانە جوان
- ڕێنمایی خۆکارانە گرتنەبەر

---

### 🎯 چۆن ئەجێنت کار دەکات

```
1. خوێندەرە پرسیار دەکات
   ↓
2. ئەجێنت پڕۆسێسی پرسیار دەکات
   ↓
3. بابەت پەیدا دەکات
   ↓
4. وتاری فێرکردنەوە دەدات لە سێ زمان
```

#### مثالان:

**کوردی:**
```
خوێندەر: "وتار بدە پلانەکان"
ئەجێنت: [وتاری فێرکردنەوەی سێ پلان]
```

**عەرەبی:**
```
المستخدم: "شرح الخطط"
الوكيل: [شرح مفصل للخطط الثلاث]
```

**English:**
```
User: "Tell me about plans"
Agent: [Detailed explanation of three plans]
```

## 🛠️ ستاپ کردن و رۆژنیگاری

### پێویستی:
- Node.js v16+
- npm

### دانانی:

```bash
# داخولی دایرەکتۆری پڕۆژە
cd SAS-platform

# دانانی پاکیجەکان
npm install

# رۆژنی سێرڤەر
npm start

# بۆ دێویلۆپمنتی بە جێگیری
npm run dev
```

ئینجا بکەوە: `http://localhost:3000`

## 📝 مثالی بەکارهێنانی ئەجێنت

### سەناریۆی یەکەم - بەکارهێنەری نوێ:
```
1. بکەوە ماڵپەڕە
2. زمانت هیل بکە (کوردی/عەرەبی/ئینگلیزی)
3. بە مەرج رازی بە
4. بۆ جێبەجێی شتی پەیدا بکە، ئەجێنتی خاپی بکە
5. پڕسیار بکە: "پلانەکان شنۆ بدە"
6. ئەجێنت وتاری فێرکردنەوە دەدات
```

### سەناریۆی دووەم - بەرهەمی دروستکردن:
```
1. پڕۆمپتی خۆت بنووسە
2. بۆ ئەجێنت پڕسیار بکە: "چۆن بەرهەم دروستبکەم؟"
3. ئەجێنت قۆل بە قۆل رێنمایی دەکات
```
