### 2️⃣ سیستەمی پارەدان (Payment System) ✅

**دروازەکانی پارەدان:**

#### ✅ Stripe (کریدی کارت)
```
کریدی کارت → CVV → Expiry → بڕێ!
```

#### ✅ FastPay (SMS Billing)
```
📱 ژمارەی تەلەفۆن → SMS تیکت → پارە تێکڕاو
```

**پڕۆسەی پارەدان:**
1. بۆتنی "🛒 کڕین" 
2. مۆدالی پارەدان دادێ
3. هەڵبژاردنی دروازە (کریدی/FastPay)
4. دروستکردنی کلیلی ئەکتیڤکردن
5. نیشاندانی پێتی سەرکەوتوویی
